# 🍎 拍一拍就能知道热量？AI食物分析神器来啦！

每天纠结"今天吃什么"？不知道外卖多少卡路里？想对比两种食物哪个更健康？😱

今天给大家安利一个超实用的**AI食物热量分析工具**，用AI帮你轻松搞定饮食管理！✨  [更多作品](PROJECT.md)

## 🎯 这是什么神器？

一个基于 React + Ant Design 开发的现代化食物营养分析平台，集成AI多模态模型，上传食物图片或输入描述，3秒内获取详细营养报告！

## 💡 核心功能

✅ **AI视觉识别** - 拍照上传食物图片，AI自动识别
   - 🍜 智能识别食物类型
   - 📊 精准计算热量和营养素

✅ **多维度营养分析** - 六大营养指标全覆盖
   - 热量、蛋白质、脂肪、碳水
   - 提供科学配比建议

✅ **个性化饮食建议** - 根据营养数据给出健康建议
   - 搭配蔬菜增加纤维
   - 控制食用份量
   - 选择健康饮品

✅ **食物对比功能** - 一键对比两种食物
   - 热量对比
   - 营养成分对比
   - 帮你选择更健康的食物

✅ **历史记录管理** - IndexedDB本地存储
   - 永久保存分析记录
   - 随时查看历史数据
   - 支持删除记录

✅ **灵活输入方式**
   - 上传图片识别
   - 文字描述分析
   - 图片+描述组合

## 🚀 为什么选择这个工具？

- 1️⃣ **AI驱动** - 集成硅基流动DeepSeek-VL2大模型，识别更准确
- 2️⃣ **本地存储** - IndexedDB保证数据不丢失
- 3️⃣ **响应式设计** - 手机/平板/电脑都能用
- 4️⃣ **隐私保护** - 数据仅保存在本地浏览器
- 5️⃣ **零配置部署** - 阿里云ESA Pages一键上线

## 📖 使用方法

- 1️⃣ 打开工具网页
- 2️⃣ 选择输入方式（上传图片/输入描述）
- 3️⃣ 等待AI分析完成
- 4️⃅ 查看详细营养报告和建议
- 5️⃅ 保存到历史记录或进行对比

就这么简单！🎉

## 🛠️ 技术栈

- **前端框架**: React 18
- **路由管理**: React Router DOM
- **UI组件库**: Ant Design 5.x
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **AI模型**: 硅基流动 DeepSeek-VL2
- **本地存储**: IndexedDB
- **部署平台**: **阿里云 ESA Pages** 🚀

## 💰 为什么要做这个？

作为程序员，深知健康饮食的重要性！市面上要么是收费App，要么是功能简陋的网页版。所以决定自己做一个真正好用的AI营养分析工具，免费分享给所有人，帮助大家科学饮食！

## 📁 项目结构

```
src/
├── App.jsx              # 主应用组件
├── index.jsx            # 入口文件
├── index.css            # 全局样式
├── pages/               # 页面组件
│   ├── Home.jsx         # 首页
│   ├── AnalysisResult.jsx  # 分析结果
│   ├── Compare.jsx      # 对比页面
│   ├── History.jsx      # 历史记录
│   └── Settings.jsx     # 设置页面
├── api/                 # API服务
│   └── siliconflow.js   # 硅基流动API
└── utils/               # 工具函数
    ├── indexedDB.js     # IndexedDB封装
    └── historyDB.js     # 历史记录管理
```
