# AI食物热量分析系统

AI食物热量分析系统是一个智能化的食物识别与营养分析平台，通过AI技术分析食物图片或描述，提供热量信息、营养成分分析及健康饮食建议。

## 功能特性

- **首页 (Home)**: 应用主界面，提供食物分析入口
- **分析结果 (AnalysisResult)**: 展示食物热量和营养分析结果
- **对比 (Compare)**: 对比不同食物的营养信息
- **历史记录 (History)**: 查看历史分析记录
- **响应式设计**: 完美适配桌面、平板和手机设备

## 技术栈

- **前端框架**: React 18
- **路由**: React Router DOM
- **UI组件库**: Ant Design
- **构建工具**: Vite
- **样式**: Tailwind CSS + 自定义样式
- **图标**: Ant Design Icons

## 项目结构

```
src/
```

## 路由

- `/` - 首页
- `/compare` - 对比页面
- `/history` - 历史记录页面
- `/analysis/:id` - 分析结果详情页面
